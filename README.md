# bahnhof

### full stack application

##### frontend: react js

##### backend: express js

## running the Application :

### 1. Clone the repo

```sh
> git clone git@github.com:wafy89/Bahnhof.git

```

### 2. Install NPM packages

```sh
> cd bahnhof
> npm run install-deps
```

### 3. run application

```sh
> npm start
```
